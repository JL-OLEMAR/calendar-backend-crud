# **Calendar Backend**

## npm install dependencies

```bash
npm install
```

## run API

```bash
npm run dev
```

## run Linter with ***Standard***

```bash
npm run lint
```

### crear .env en la raiz del proyecto sin <>
- PORT=`4000`
- DB_CNN=`mongodb+srv://<user>:<password>@<clusterName>.ecs5b.mongodb.net/<bdName>`
- SECRET_JWT_SEED=`<palabra secreta>`
